File Duplicate Finder
Проста консольна утиліта на C# для пошуку дублікатів файлів у вказаній директорії за допомогою хешування SHA-256.
🚀 Як це працює
Програма сканує всі файли у вибраній папці (включаючи підпапки) і порівнює їхній вміст:
Користувач вводить шлях до папки.
Програма генерує унікальний хеш для кожного файлу.
Якщо два файли мають однаковий хеш — вони ідентичні, і програма виводить шлях до дубліката.
🛠 Технології
C# / .NET
System.Security.Cryptography (для обчислення SHA-256)
System.IO (для роботи з файловою системою)
📋 Використання
Запустіть програму.
Введіть повний шлях до папки (наприклад, C:\Users\Admin\Documents).
Перегляньте список знайдених дублікатів у консолі.
⚠️ Примітки
Програма ігнорує файли, до яких немає доступу (наприклад, системні файли під захистом).
Пошук здійснюється рекурсивно по всіх вкладених папках.
